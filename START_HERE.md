# 🚀 Quick Start Guide

## Get Your Project Tracker Running in 2 Steps!

### Step 1: Start the Backend Server

Open a terminal and run:

```bash
cd backend
npm start
```

**Expected Output:**
```
Server running on port 5000
MongoDB Connected
```

✅ If you see this, your backend is ready!

❌ If you see errors:
- Run `npm install` first (only needed once)
- Check that you have Node.js installed
- See `BACKEND_SETUP.md` for troubleshooting

### Step 2: Open the Frontend

Simply open `index.html` in your web browser:
- Double-click the file, OR
- Right-click → Open with → Your browser, OR
- Use a local server (Live Server extension in VS Code)

## ✨ You're Done!

The app should now be fully functional. Try:
1. Click the **+** button to add a project
2. Use the **✨ Generate with AI** button for smart descriptions
3. Edit, delete, and manage your projects
4. Filter and sort your project list
5. Toggle between light and dark themes

## 🐛 Troubleshooting

### "ERR_CONNECTION_REFUSED" in console?
→ Your backend isn't running. Go back to Step 1.

### "Failed to load projects"?
→ Check that MongoDB Atlas is accessible (you're using cloud database)

### Still having issues?
→ Open browser console (F12) and check for errors
→ See `FIXES_APPLIED.md` for detailed debugging info
